# Python Log File Analyzer

A Python-based CLI tool for analyzing system log files using customizable keyword filters.  
Designed to practice cybersecurity fundamentals such as log analysis, file parsing, and Linux-based tooling.

---

## Features

- Interactive command-line interface (menu-driven)
- Supports dynamic keyword filters (add/remove at runtime)
- Reads and analyzes log files line-by-line
- Counts occurrences of user-defined severity keywords (e.g., error, warning, critical)
- Generates a results report saved to an output file
- Designed for Linux environments

---

## Technologies Used

- Python 3
- File I/O
- Dictionaries & Lists
- Exception Handling
- Linux CLI workflow
